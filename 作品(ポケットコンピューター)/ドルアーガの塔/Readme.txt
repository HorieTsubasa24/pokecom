・実行方法
　g800.exe　を起動して、
　RUN Enterキー
　と入力するとゲームが開始できます。
　OFFキー左のANSキーを押すとゲームを終了します。
・操作方法
　以下に記述しています。


プログラムの実行は、ポケコンをオールリセットし
TEXTボタン
S
F
転送速度を最適なものに設定して  returnキー
CLS

BASIC画面で
MON returnキー
USER 61FF returnキー

転送ソフトやエミュレータを使って

R returnキー
しばらく待って
INFO 0100-6192
と表示されればG100?やBASIC画面でCALL256 returnキー でゲーム開始。

g800エミュレータを使う場合は
F4(menu)
S(sio)
I(in)
ファイル選択(tabキーでカレントディレクトリをひとつ遡れる)
enterキーで目的のファイルを選択してからマシン語モニタのRコマンドを実行。


ドルアーガの塔のポケコン版です。なかなか休みがなく、作る時間がなかったので、
結果的に1年近くの期間をかけることになりました。(集中して作れたならば2ヶ月ほどか。)


操作方法は、
上下左右キーでそれぞれ、上下左右に移動、スペースキーで剣を出す。

各階には宝箱があり、それを取得しながらギル(主人公)を
パワーアップさせながら最上階60階を目指し、ヒロインのカイを
助けに行く物語です。宝箱を出すにはは、各階ごとの条件をクリアする必要があり、
なかなか個性的です。

このポケコン版は、MAPがFC版、宝箱はAC版(一部不要なアイテムや
再現不能のものは省いてある)に合わせてあります。
