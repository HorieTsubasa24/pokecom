・起動方法
　g800.exe　を起動して、
　RUN Enterキー
　と入力するとゲームを開始できます。
　右上のONキーを押すとゲームが終了します。
・操作方法
　方向キーの押す方向に数字が移動します。
　数字が
　1234
　5678
　9ABC
　DEF
　の順番になればゲームクリアです。

プログラムの実行は、ポケコンの容量を空けて
TEXTボタン?
S
F
転送速度を最適なものに設定して
CLS

BASIC画面で
MON?
USER 23F?

転送ソフトやエミュレータを使って

R?
しばらく待って
INFO 0100-023F
と表示されればG100?やBASIC画面でCALL256?でゲーム開始。


g800エミュレータを使う場合は
